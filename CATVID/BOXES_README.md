# Система ящиков для получения скинов

## Что реализовано:

### 1. Модели данных:
- **Box** - модель ящика (название, описание, цена, изображение)
- **BoxDrop** - возможные выпадения из ящика (тип предмета, значение, шанс выпадения, редкость)
- **BoxOpening** - история открытий ящиков игроком

### 2. Логика работы:
- Скины теперь можно получить **ТОЛЬКО** из ящиков
- Ящики покупаются за монеты на главной странице
- При открытии ящика случайным образом выпадает один из предметов согласно настроенным шансам
- Возможные награды: скины, монеты, клики
- Редкие предметы имеют специальную анимацию

### 3. Пользовательский интерфейс:
- Секция ящиков на главной странице
- Карточки ящиков с описанием и списком возможных наград
- Кнопки покупки с проверкой достаточности монет
- Уведомления о результатах открытия
- Специальные эффекты для редких предметов

### 4. Тестовые данные:
Созданы два ящика:

**Обычный ящик (100 монет):**
- 40% - 50 монет
- 30% - 100 кликов  
- 15% - скин skin3.png
- 10% - скин skin4.png
- 5% - скин skin5.png (редкий)

**Премиум ящик (300 монет):**
- 30% - 200 монет
- 25% - 300 кликов
- 20% - скин skin5.png
- 15% - скин skin6.png (редкий)
- 8% - скин skin7.png (редкий)
- 2% - скин skin8.png (редкий)

## Как использовать:

1. Зайдите на главную страницу игры
2. Прокрутите вниз до секции "Ящики со скинами"
3. Выберите ящик и нажмите кнопку покупки
4. Получите случайную награду!

## Администрирование:

Через Django Admin можно:
- Создавать новые ящики
- Настраивать содержимое ящиков
- Изменять шансы выпадения
- Просматривать историю открытий

## Файлы системы:

- `models.py` - модели Box, BoxDrop, BoxOpening
- `views.py` - функции buy_box() и open_box()
- `admin.py` - админ-интерфейсы
- `urls.py` - URL для покупки ящиков
- `index.html` - интерфейс ящиков
- `style.css` - стили для ящиков
- `create_boxes.py` - скрипт создания тестовых данных
